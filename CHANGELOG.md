# Changelog

## 1.0.0 - 2026-07-05

Initial stable release baseline.

- macOS menu bar client for AnyConnect-compatible OpenConnect connections.
- Local SOCKS5 mode through `openconnect --script-tun` and `ocproxy`.
- Multiple simultaneous profiles with separate SOCKS ports.
- Touch ID-protected credential vault with explicit unsafe local storage opt-in.
- Per-profile auto-start, reconnect handling, connection history, and route import viewer.
- Packaged `.app` release flow with bundled runtime binaries and app icon.
